Design Flow
RTL → Logic Synthesis → Place & Route → GDSII → Tape-out. Tools: Cadence Virtuoso / Synopsys Design Compiler
Power Techniques
Clock gating reduces dynamic power by ~30%. DCVSL dynamic logic cuts glitch power. Operand isolation prevents unnecessary switching.
Layout Tools
Microwind / Magic VLSI for physical design. DRC/LVS checks mandatory. Standard cells from 180nm PDK library.
